# Expansión Estratégica de Biogenesys

Proyecto de análisis de datos realizado con Python y Power BI para identificar ubicaciones óptimas de expansión de laboratorios farmacéuticos en Latinoamérica.

Herramientas: Python (pandas, NumPy, matplotlib, seaborn), Power BI.

Incluye:
- Limpieza y exploración de datos (EDA)
- Integración de fuentes (ETL)
- Visualizaciones en python con Matplotlib y Seaborn 
- Visualización de métricas y KPIs en Power BI
